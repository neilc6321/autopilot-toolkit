# autopilot-toolkit

A collection of upstream skills and autopilot workflow skills, deployed to `~/.agents/skills/`.

## Purpose

- **`skills/upstream/`** — upstream skills (managed via git subtree)
- **`skills/autopilot/`** — autopilot workflow skills

## Compatibility

Target: **reasonix/codex** compatible.

## Install

```bash
bash install.sh
```
